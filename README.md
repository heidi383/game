
<영화 맞추기 게임>\
제한시간 40초 동안 16개의 박스로 가려져 있는 이미지를 보고 영화 제목을 맞춘다.
박스를 클릭하면 사라지고 이미지의 일부분이 보인다. 
박스를 클릭할 때마다 점수가 한 점씩 내려간다.
